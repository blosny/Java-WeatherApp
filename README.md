# Java Hava Durumu Uygulaması (Swing GUI)

Java Swing ile geliştirilmiş, kullanıcı dostu ve şık bir masaüstü hava durumu uygulamasıdır. API entegrasyonu sayesinde dünya çapındaki şehirlerin güncel hava koşullarını anlık olarak sunar.

## Özellikler
- **Küresel Erişim**: API desteği ile dünya genelindeki tüm şehirlerin hava durumunu öğrenebilirsiniz.
- **Kapsamlı Veriler**: Sadece sıcaklık değil; yağış, nem, rüzgar hızı ve güncel hava durumunu gösteren özel ikonlar içerir.
- **Kullanıcı Dostu Arayüz**: Şehir adı girildiğinde hızlıca sonuç veren, modern tasarımlı bir GUI.
- **Hızlı Arama**: Şehir adını yazdıktan sonra klavyedeki **'Enter'** tuşunu kullanarak anında arama yapabilirsiniz; bu da uygulamayı daha etkili ve hızlı hale getirir.

## Teknik Detaylar
- **Dil**: Java
- **Kütüphane**: Swing (GUI)
- **Veri Kaynağı**: Hava Durumu API'si

## Kurulum
1. Bilgisayarınızda **JDK 8 veya daha üstü** bir sürümün yüklü olduğundan emin olun.
2. Depoyu klonlayın:
   ```bash
   git clone https://github.com/blosny/Java-WeatherApp.git
   ```
3. Projeyi bir Java IDE'sinde açın ve `src/App.java` dosyasını çalıştırın.

## Kullanım
1. Uygulama açıldığında arama alanına bir şehir adı (örn: Istanbul, London) yazın.
2. Klavyenizden **Enter** tuşuna basın veya arama ikonuna tıklayın.
3. Ekrandaki ikonlar ve metrikler üzerinden güncel hava durumunu analiz edin.
